# 🐍 Day 03 - Python Strings

## 📚 Topics Covered

### 1. Escape Sequences
Learned how to use special characters inside strings.

Examples:
- `\n` → New Line
- `\t` → Tab Space
- `\\` → Backslash
- `\"` → Double Quotes

---

### 2. f-Strings
Learned how to format strings in a simple and readable way.

Example:

```python
name = "Roshan"
age = 20

print(f"My name is {name} and I am {age} years old.")

### 3. String Indexing

Accessing individual characters using their position.

Example:

text = "Python"

print(text[0])   # P
print(text[-1])  # n
### 4. String Slicing

Extracting a portion of a string.

Example:

text = "Python"

print(text[0:3])   # Pyt
print(text[2:6])   # thon
### 5. String Operations

Practiced common string operations:

Concatenation (+)
Repetition (*)
Membership (in)
Length (len())

Example:

first = "Hello"
second = "World"

print(first + " " + second)
